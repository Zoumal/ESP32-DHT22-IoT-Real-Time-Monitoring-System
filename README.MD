Project Statements

1. Developed a real-time temperature and humidity monitoring system using an ESP32 microcontroller and DHT22 digital sensor.

2. Implemented FreeRTOS multitasking to manage sensor acquisition, WiFi communication, and embedded web server operations simultaneously.

3. Designed and integrated an HTTP web server that allows live environmental data monitoring through any browser connected to the same network.

4. Improved system reliability by implementing timing optimization, retry logic, checksum validation, and stable GPIO communication using a 10kΩ pull-up resistor.

5. Applied modular embedded software engineering practices by separating the project into multiple `.c` and `.h` files for improved scalability, debugging, and maintainability.
